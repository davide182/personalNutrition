# Progetto Nutrizionisti

Sistema di prenotazione visite nutrizionali basato su architettura distribuita.

## Architettura

- **Frontend:** React + TypeScript + Vite + Shadcn UI
- **Backend principale:** Spring Boot 3.x (Java 21) - validazione e persistenza
- **Backend secondario:** Flask (Python 3.12) - servizi specifici
- **Database principale:** PostgreSQL 16
- **Database log:** MongoDB
- **DevOps:** Docker + Docker Compose + GitHub Actions

## Struttura del progetto

- `frontend/` - applicazione React
- `backend-spring/` - backend principale Spring Boot
- `backend-flask/` - backend secondario Flask
- `docker/` - configurazioni Docker
- `docs/` - documentazione tecnica
